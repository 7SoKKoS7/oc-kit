# oc-kit

Personal automation kit for macOS — experiments, install scripts, AI agent helpers.

## Contents

- `setup/` — generic install scripts
- `experiments/` — sandbox stuff
- `examples/` — examples
- `docs/` — notes
- `x/` — per-machine overlays

## Usage

```bash
curl -fsSL https://raw.githubusercontent.com/7SoKKoS7/oc-kit/main/setup/bootstrap.sh | bash
```

Sandbox / experimental — use at your own risk.
